## Getting Started

```bash
git clone https://github.com/Happy-Elders/daycare-portal.git
npm install
npm i -g next
npm i next-transpile-modules
```

## Environment Variables

add a file
### .env.local
```bash
NEXTAUTH_URL=
NEXTAUTH_SECRET= go to https://generate-secret.now.sh/32

FACEBOOK_ID=
FACEBOOK_SECRET=

LINE_ID=
LINE_SECRET=

GOOGLE_ID=
GOOGLE_CLIENT_SECRET=
```



## Run

```bash
npm run dev
```
